## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:
  * A basic understanding of Ruby and Git
### Usage
1. Install Ruby
2. Clone this repo 
```
git clone https://github.com/mershark/ruby-group-capstone.git
```
```
cd ruby-group-capstone
ruby main.rb
```

### Run RSpec Tests

```
 rspec spec
```
