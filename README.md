# 📖 Catalog of My Things <a name="about-project"></a>

**Catalog of My Things:** This application is designed as a console utility enabling users to maintain a catalog of their possessions, encompassing a variety of categories such as books, music albums, movies, and games. The entire system is structured according to this [UML](https://github.com/microverseinc/curriculum-ruby/blob/main/group-capstone/images/catalog_of_my_things.png) class diagram

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:
  * A basic understanding of Ruby and Git
### Usage
1. Install Ruby
2. Clone this repo 
```
git clone https://github.com/mershark/ruby-group-capstone.git
```
```
cd ruby-group-capstone
ruby main.rb
```

### Run RSpec Tests

```
 rspec spec
```
